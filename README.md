# Tile Map Example Code for ZX Spectrum Next

Tile Map Demo

- A simple demo showing the use of ZX Spectrum Next hardware Tile Map Feature to demonstrate drawing a Manic Miner level, Central Cavern
- Tile Map examples are static, 40x32 size, with and without attribute bytes, which can be selected via changing commented code
- ZX Spectrum Next's relevants ports are initialized, then a tile map and it's defintions are loaded into memory
- Code compiled using SJASMPLUS 1.13.3
- .nex file runs on CSPECT emulator
- Thanks to the ZX Spectrum Next development scene who have helped with advice and code
